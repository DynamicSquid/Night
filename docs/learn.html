Hello World!

For basic output to the console, you can use the print() function. It takes a single parameter, and displays it to the console. Singleline comments are also supported.

print("Hello World!\n")

# this is a comment

print("Enjoy the tutorial!")

Variables

To initialize a variable, you use the keyword set. Note that, just like most interpreted language, Night's variables does not have the value type strongly declared. Instead, it changes the type to the currently assigned value, like this:

set int = 12

set float = 12.5

set str = "12stuff"

# must be confirmed if this exist at all
set char = 't'

set array = [ 1, 2 ]

array = 22

Notice that the array variable value type can be changed. Although as a matter of fact, this is actually a bad practice, but possible nonetheless.

User Input

User input in Night takes the form of the function input().

print("What is your age? ")
set age = input()

Expressions

Common boolean, arithmetric, and string expressions are supported. They can be used whenever a value is expected.

Some implicit conversions are also supported. Type int and type float can be used interchangeably, and any type can concatenate into type str.

Conditionals

Conditionals in Night take use of curly brackets to group code together.

set animal = "squid"

if (animal == "squid")
{
 print("Hi squid friend!")
}

Functions

Functions in Night support return value of all six types, along with a null (void) type. null type functions just means that the functions don't have a return value.

def add(a, b)
{
 print('Adding two numbers: ' + a + ' and ' + b + '\n')
 return a + b
}

print(add(2, 3))

Note that functions have to be defined before they are used.

Loops

There are two types of loops in Night, the while and for loop. The while loop will only run if its condition evaluates to true, and the for loop will only run a prespecified amount of times.

Arrays

To initialize an array, assign a variable to the array literal, basically elements with value in a square bracket container. Arrays could possibly contain one more than type of value.
